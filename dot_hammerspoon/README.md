## Spoons

I am using the following [Spoons](https://github.com/Hammerspoon/hammerspoon/blob/master/SPOONS.md#what-is-a-spoon):

### [Headspace.spoon](https://github.com/evantravers/Headspace.spoon)

Headspace lets you transform your computer usage into a more focused "monotasking" environment. [evantravers](https://github.com/evantravers) explains it [here](https://evantravers.com/articles/2023/02/20/raycast-shortcuts-headspace/).

## Lua Scripts:

### [slowq.lua](https://github.com/dbmrq/Dotfiles/blob/master/Hammerspoon/.hammerspoon/slowq.lua)

SlowQ lets you hold Command + Q for a while before closing an app, so you won't do it accidentally. Replaces apps like CommandQ and SlowQuitApps.
